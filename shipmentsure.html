ShipmentSure

An intelligent shipment delay prediction platform that combines Machine Learning, Explainable AI, and interactive analytics to help logistics teams identify delivery risks, improve operational efficiency, and support proactive decision-making.

PythonFlaskXGBoostSHAPSQLiteSeaborn

Project Overview

ShipmentSure is an end-to-end logistics analytics platform developed to predict whether a shipment will be delivered on time using historical shipment and supplier data. The application combines machine learning, data engineering, and Explainable AI to provide accurate delay predictions while helping users understand the factors influencing each prediction. By delivering real-time insights through an interactive web application, ShipmentSure enables logistics teams to reduce operational risks, improve planning, and enhance customer satisfaction.

Data Analysis & Engineering

The project began with extensive exploration of shipment and supplier data containing attributes such as shipment mode, customer ratings, product importance, product cost, prior purchases, discount offered, warehouse location, and delivery status. Careful data preprocessing and feature engineering were performed to improve model performance and generate meaningful business insights.

  • Exploratory Data Analysis: Conducted comprehensive univariate, bivariate, and correlation analysis to identify patterns affecting shipment delays. The analysis revealed that higher discount percentages, shipment modes, and product characteristics significantly influenced delivery performance.
  • Feature Engineering: Created new features like Cost_to_Weight_Ratio and High_Discount indicator. Categorical features were encoded and numerical ones normalized.

Model Development

Multiple machine learning models were trained and evaluated, including Logistic Regression, Decision Trees, Random Forests, Naive Bayes, SVM, LightGBM, and CatBoost.

  • Final Model: XGBoost was selected as the final model, achieving the highest accuracy of 94% with robust performance against class imbalance and feature correlations.
  • Explainable AI: Integrated SHAP (SHapley Additive exPlanations) to interpret model predictions, providing stakeholders with clear insights into which factors (like discount or product weight) caused a delay.

Deployment & Web Application

The model was deployed as a fully functional web application using Flask, hosted on the Render Cloud Platform.

  • Interactive Dashboard: Features a secure, role-based login system (SQLite) and real-time shipment delay prediction inputs.
  • Automated Alerts: Configured an SMTP integration to send automated email notifications to managers whenever a shipment is predicted to be delayed.
  • Map Visualization: Incorporated Folium to display interactive shipment route maps.
